# DriverTime

System do zbierania i analizy czasu pracy kierowców.

## Funkcje

- Zarządzanie firmami
- Zarządzanie kierowcami
- Zarządzanie pojazdami
- Import plików DDD
- Analiza zgodności z UE 561/2006
- Raporty PDF i Excel
- Dashboard
- Powiadomienia

## Technologia

- ASP.NET Core
- PostgreSQL
- React
- Docker

Cel projektu

Aplikacja dla wielu firm transportowych umożliwiająca:

* zarządzanie kierowcami,
* zarządzanie pojazdami,
* import plików DDD,
* analizę zgodności z UE 561/2006 i AETR,
* raportowanie naruszeń,
* zarządzanie dokumentami kierowców,
* dashboard i statystyki.
