# DriverTime

System do zbierania i analizy czasu pracy kierowców.

## Funkcje

- Zarządzanie firmami
- Zarządzanie kierowcami
- Zarządzanie pojazdami
- Import plików DDD
- Analiza zgodności z UE 561/2006
- Raporty PDF i Excel
- Dashboard
- Powiadomienia

## Technologia

- ASP.NET Core
- PostgreSQL
- React
- Docker
