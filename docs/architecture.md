DriverTime Architecture

Project Goal

DriverTime is a platform for transport companies that manages drivers, vehicles and tachograph data.

Main Modules

* Companies
* Users
* Drivers
* Vehicles
* Driver Documents
* DDD Import
* Driver Activities
* Violations
* Reports
* Notifications

Backend

Technology:

* ASP.NET Core
* Entity Framework Core
* PostgreSQL

Layers:

* API
* Application
* Domain
* Infrastructure

Frontend

Technology:

* React

Security

* JWT Authentication
* Role-based Authorization

Multi-company Support

Every business entity contains CompanyId.

Data is isolated between companies.

Future Modules

* DDD Parser
* Violation Engine
* PDF Reports
* Excel Reports
* Fleet Dashboard
